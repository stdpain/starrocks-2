# StarRocks

import FeatureList from '/src/components/Features/index.js'

StarRocks 是一款新一代高性能分析型数据仓库，支持实时、多维、高并发的数据分析。 StarRocks 采用 MPP 架构，并配备了全向量化执行引擎、支持实时更新的列式存储引擎，以及由完全自定义的基于成本的优化器 (CBO) 、智能物化视图等组成的一系列丰富的功能。 StarRocks 支持从各种数据源进行实时和批量数据摄取。它还允许您直接分析存储在数据湖中的数据，而无需 0数据迁移。

StarRocks 兼容 MySQL 协议，可以使用 MySQL 客户端和流行的 BI 工具轻松连接。 StarRocks 具有高可扩展性、高可用性且易于维护。它在业界被广泛采用，为各种 OLAP 场景提供支持，例如实时分析、即席查询、数据湖分析等。
